# CONCEPTS — five coming-soon pages across three directions

Each concept is a complete coming-soon landing page in a different aesthetic direction. The owner reviews screenshots and picks one. The winner goes live at zone2hp.com and later grows into the full site (phase-2 IA at the end of this file). All five share the same design system and the same required elements, and all five must pass `COMPLIANCE.md`.

> **Status: resolved.** Concept **A2** (The Film direction) was chosen and is live (`src/pages/index.astro`). A1 remains at `src/pages/concepts/a1.astro` for reference; B1, C1 and C2 were not built. The phase-2 IA at the end of this file still guides the full site.

## Required on every concept

- Wordmark (ZONE TWO, per `DESIGN-SYSTEM.md`).
- An opening-soon statement, for example `Opening soon in Eastwood.`
- A one-line summary of services: `CHIROPRACTIC · REHAB · EXERCISE PHYSIOLOGY · DRY NEEDLING`.
- Address with an embedded map: `Suite 104 / 2 Rowe St, Eastwood, NSW 2122`.
- Phone (placeholder) and email (`info@zone2hp.com`) as `tel:` and `mailto:` links.
- A footer.

Optional shared touches: the line `Recover. Rebuild. Perform.`, a short scroll, a scroll cue on the video-led concepts.

### Map embed

Use a standard embedded map of the Eastwood address. Keep it light and on-brand: a bordered map block with the written address beside it, not a heavy coloured pin-cluster. If a tinted or minimal map style is wanted, note it as a later refinement; for the coming-soon page a clean standard embed is fine. Make sure it is keyboard reachable and has an accessible label.

## The three directions and five pages

### Direction A — The Film (cinematic, Nivis-like)

Near full-bleed hero loop, minimal chrome, the wordmark and opening-soon line over the visual, a scroll cue, then a compact info band (services line, map, contact) below. Maximum visual impact, least text. This is the most direct answer to the owner's "wow visual with video" ask.

- **A1 — Snow / blizzard hero.** Cool alpine descent. Headline option: `Stay in the season.` Tests the snow and ski vibe.
- **A2 — Night trail hero.** Headlamp on a dark trail. Headline option: `Every session is base miles.` Tests the night-running and trail vibe.

A1 and A2 are the same skeleton with different footage and grade, so the owner can react to which sport energy fits the brand. This is a cheap, high-signal A/B for them.

### Direction B — The Zones (lookbook, Quechua-like)

A short chaptered scroll. Three brief chapters titled `Recover`, `Rebuild`, `Perform`, each a calm environment still with a one-line of factual copy and a Quechua-style coordinate label, resolving into the info block at the end. Tests whether the owner likes an immersive scroll even for a holding page.

- **B1 — Chaptered scroll.** Eyebrow coordinate motif: `EASTWOOD 33.79° S, 151.08° E` (verify exact coordinates). Keep it short, this is still a coming-soon page, not the full site.

### Direction C — The Method / Console (Myo structure, spec-sheet styling)

Borrows Myodetox's clean structure but compressed to a coming-soon page and styled as a technical spec sheet. Wordmark and statement, a tight services list presented as labelled rows, the info block. Two variations test how human versus technical the brand should feel.

- **C1 — Editorial warm.** Type-led, generous white space, stone palette, a single quiet still, maybe a refined serif for one headline. The calmer, more human take.
- **C2 — Technical console.** Mono accents, spec-sheet rows, cooler chrome palette, the services and details laid out like a data panel. The hardest-technical take, leaning fully into the Zone Two idea.

1. Hero (same as the coming-soon, video now live).
2. `Recover. Rebuild. Perform.` statement.

6. Location and map, hours, contact.
7. Call to action (book or call, depending on whether booking is live).

No testimonials section, ever. See `COMPLIANCE.md`.
